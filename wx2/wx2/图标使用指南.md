## 使用方法

### 1. Font Class 方式（推荐）

这是最常用和推荐的方式，在Vue组件中使用：

```vue
<template>
  <!-- 基本用法 -->
  <span class="icon lishuai-dianhua"></span>
  
  <!-- 结合样式使用 -->
  <view class="phone-icon">
    <span class="icon lishuai-dianhua"></span>
    拨打电话
  </view>
</template>

<style>
.phone-icon {
  display: flex;
  align-items: center;
  color: #5cb85c;
  font-size: 18px;
}

/* 自定义图标样式 */
.icon {
  margin-right: 8rpx;
}
</style>
```

### 2. 使用 uni-icons 组件

你的项目中已经在使用 `uni-icons` 组件来显示图标：

```vue
<template>
  <!-- 使用uni-icons组件 -->
  <uni-icons 
    custom-prefix="icon" 
    type="lishuai-dianhua" 
    size="18" 
    color="#5cb85c">
  </uni-icons>
  
  <!-- 定位图标示例 -->
  <uni-icons 
    custom-prefix="icon" 
    type="lishuai-dingwei" 
    size="12" 
    color="#8a8a8a">
  </uni-icons>
  
  <!-- 删除图标示例 -->
  <uni-icons 
    color="#999999" 
    custom-prefix="icon" 
    type="lishuai-shanchu" 
    size="18">
  </uni-icons>
</template>
```

### 3. Unicode 方式

直接使用Unicode编码：

```vue
<template>
  <!-- 电话图标 -->
  <span class="icon">&#xe65a;</span>
  
  <!-- 定位图标 -->
  <span class="icon">&#xe8c4;</span>
</template>

<style>
.icon {
  font-family: "icon";
  font-size: 16px;
  color: #333;
}
</style>
```

## 注意事项

1. **必须引入CSS文件**: 确保在 `App.vue` 中已经引入了 `@import "@/style/iconfont/iconfont.css";`

2. **正确的类名**: 所有图标类名都以 `lishuai-` 开头，这是项目的自定义前缀

3. **兼容性**: 图标字体支持所有现代浏览器和小程序平台

4. **性能优化**: Font Class 方式比 Unicode 方式更易维护，比 SVG 方式性能更好

## 添加新图标

如果需要添加新图标：

### 步骤1：登录 IconFont 管理后台
访问你的图标项目：[IconFont项目页面](https://www.iconfont.cn/manage/index?manage_type=myprojects&projectId=4857955)

### 步骤2：添加图标到项目
- **方式一**：从图标库搜索添加
  1. 在 IconFont 首页搜索图标
  2. 点击"添加入库"选择你的项目
- **方式二**：上传自定义图标
  1. 在项目页面点击"上传图标"
  2. 上传 SVG 格式文件

### 步骤3：重新生成和下载字体
1. 点击"Font class"标签
2. 点击"下载至本地"
3. 解压下载的文件包

### 步骤4：替换项目文件
需要替换 `style/iconfont/` 目录下的以下文件：
- `iconfont.css` (最重要)
- `iconfont.woff2`
- `iconfont.woff` 
- `iconfont.ttf`
- `iconfont.json`

### 步骤5：更新图标使用指南
在新增图标后，建议更新本文档的图标列表，添加新图标的信息。

## 最佳实践

1. **语义化使用**: 根据功能选择合适的图标
2. **统一风格**: 保持项目中图标的大小和颜色风格一致
3. **性能考虑**: 优先使用 Font Class 方式
4. **备用方案**: 为重要功能提供文字备用方案
